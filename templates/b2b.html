Dashboard B2B Spec Tech

AI回答の品質を、
自動で監視・制御する。

RECO3は、LLM出力の信頼性を数値化し、危険な回答を自動で検知・抑制するミドルウェアです。

導入のご相談

課題

回答品質が不安定

LLMの出力は毎回異なり、誤情報や断定的な回答が混在する。人手でのチェックはスケールしない。

監査ログがない

いつ・誰に・何を回答したか。BtoB運用では説明責任が求められるが、ログ基盤がない。

暴走を止められない

異常な出力が連続しても検知できず、ユーザーにそのまま届く。フェイルセーフが存在しない。

RECO3の解決策

Input Gate

入力の曖昧度・感情圧力・非現実性を数値化。危険な入力を事前にスコアリング。

Output Gate

LLM回答の断定密度・エビデンス不足・内部矛盾を検出。品質スコアを自動付与。

自動制御ループ

スコアが閾値を超えると自動で再生成。クールダウンとフェイルセーフで暴走を防止。

監査ログ

全セッションのスコア・verdict・フィードバックを記録。APIで取得可能。

主要機能

01

リアルタイム監視

3秒間隔でシステム状態をポーリング。異常を即座に検知。

02

自動トリガ制御

条件一致時のみLLMを呼び出し。60秒クールダウン、連続失敗3回で自動停止。

03

Dual LLM対応

OpenAI GPT / Anthropic Claude を切り替え可能。自動選択モードあり。

04

PWA対応

iOS/Android/デスクトップにインストール可能。オフラインでもUIが動作。

デプロイ

セキュリティ

サーバーサイド認証

APIキーはサーバー側のみ。フロントに秘密鍵を埋め込まない。セッションCookieで認証。

監査証跡

全リクエスト/レスポンスのスコア・判定結果をログ。コンプライアンス対応。
